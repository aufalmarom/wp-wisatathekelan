Documentation
    Setup Home Page
        -Go to Page->add new and give the name of home page and select home template for home page
        -Go to Setting->Reading-> A static page (select below) and select page what we create for home.
    
    Add Parallax Section
        -Go to Appearance -> Customizer -> Home Page Setting. Enable the section and configure the setting.
